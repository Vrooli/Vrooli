# Basic Claude Code Usage Examples

This directory demonstrates fundamental Claude Code usage patterns.

## Examples in this Directory

### interactive-session.sh
Start an interactive Claude Code session and demonstrate basic commands.

### simple-prompts.sh  
Run single prompts using the management script API for automation.

### file-analysis.sh
Analyze existing code files using Claude Code's file understanding capabilities.

### command-composition.sh
Use Claude Code with Unix pipes and command composition.

## Prerequisites

- Claude Code installed and authenticated
- Basic project with some code files
- Terminal access

## Running Examples

```bash
# Make scripts executable
chmod +x *.sh

# Run individual examples
./interactive-session.sh
./simple-prompts.sh
./file-analysis.sh
./command-composition.sh
```

## Safety Notes

These examples use read-only operations and safe editing commands. They demonstrate Claude Code's capabilities without making destructive changes to your system.
