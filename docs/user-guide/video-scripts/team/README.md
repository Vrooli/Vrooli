# Team Video Script

**Duration**: 30-45 seconds  
**Target Audience**: Users interested in collaboration and team management  
**LandingView Integration**: Corresponds to `videoUrls.Team` on teams slide  
**Thumbnail**: Team management interface showing members, agents, and shared resources

## Video Overview

This video demonstrates how human + AI teams amplify productivity through shared intelligence, collaborative workflows, and seamless resource sharing.

## Key Messages

1. **Human + AI Teams**: "3 humans + 12 AI agents = infinite possibilities"
2. **Shared Intelligence**: "Team knowledge compounds automatically"
3. **Universal Access**: "Work from anywhere - web, API, or MCP integration"

## Scene Breakdown

### **Team Amplification Demo** (0:00-45 seconds)

**Opening** (0:00-0:08)
**Visual**: Split screen - traditional team vs. AI-enhanced team
**Script**: *"Traditional team: 3 people, endless meetings. AI-enhanced team: 3 humans + 12 AI agents, continuous productivity."*

**Fast-Forward Team Setup** (0:08-0:20)
**Visual**: Time-lapse team creation with agents being added
**Script**: *"Team setup in 60 seconds - add humans and AI agents..."*
**On-screen**: "FAST-FORWARD: TEAM CREATED"
- Invite team members
- Select AI agents (Content Creator, Data Analyst, Project Manager)
- Shared workspace appears

**Collaborative Intelligence** (0:20-0:35)
**Visual**: Multi-user chat with humans and AI agents working together
**Script**: *"Shared AI sessions - humans strategize, AI agents execute:"*
```
Sarah: "Plan our Q4 campaign - $50K budget"
Content Agent: "Creating campaign themes and calendars..."
Marcus: "Add ROI tracking throughout"
Data Agent: "Building real-time performance dashboards..."
```

**Universal Access** (0:35-0:45)
**Visual**: Multiple platform access - web, API, MCP server
**Script**: *"Access team AI anywhere - browser, API, or development tools."*
**Visual**: Split screen showing web interface, code integration, IDE extension

## Production Notes

### **Fast-Forward Techniques**
- **Team Setup**: Compress setup process into 12-second time-lapse
- **Agent Addition**: Quick selection and workspace creation
- **Collaboration Flow**: Rapid back-and-forth between humans and AI
- **Platform Access**: Quick cuts showing multiple access methods

### **Visual Elements**
- **Split Screen Opening**: Traditional vs. AI-enhanced team comparison
