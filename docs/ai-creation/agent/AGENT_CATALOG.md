## Event Subscriptions

Agents subscribe to these predefined platform events:

### SWARM Events
- `swarm/started` - New swarm execution
- `swarm/state/changed` - State transitions
- `swarm/resource/updated` - Resource changes
- `swarm/config/updated` - Configuration updates
- `swarm/team/updated` - Team changes
- `swarm/goal/created` - New goals
- `swarm/goal/updated` - Goal progress
- `swarm/goal/completed` - Goal success
- `swarm/goal/failed` - Goal failure

### RUN Events
- `run/started` - Routine execution start
- `run/completed` - Routine success
- `run/failed` - Routine failure
- `run/task/ready` - Task availability
- `run/decision/requested` - User input needed
- `step/started` - Step execution
- `step/completed` - Step success
- `step/failed` - Step failure

### SAFETY Events
- `safety/pre_action` - Pre-execution validation
- `safety/post_action` - Post-execution validation
