# Codex Agent Guidelines

This repository uses the [Cursor](https://docs.cursor.sh/) format for agent documentation. The rules you must follow live under `.cursor/rules`. Read those files before working on any task.

**Setup**
- Run `bash scripts/manage.sh setup` to install dependencies and configure your environment.

**Linting**
- Run `pnpm -r run lint` at the repository root.

**Testing**
- Run `pnpm -r run test` once packages are installed.

Additional project documentation is located in the `/docs` directory.
