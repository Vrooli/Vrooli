## üöÄ Getting Started

### Prerequisites
- PostgreSQL (for tech tree data storage)
- Ollama (for AI strategic analysis)
- Node.js (for visualization interface)

### Quick Start
```bash
# Start the complete strategic intelligence system
vrooli scenario run tech-tree-designer

# View current civilization progress
tech-tree-designer status --verbose

# Get AI recommendations for next scenarios to build
tech-tree-designer analyze --resources 8 --timeline 6 --priority software,manufacturing

# Launch interactive visualization
tech-tree-designer visualize --full

# View strategic milestones
tech-tree-designer milestones
```

### Strategic Analysis Examples
```bash
# Analyze optimal development path with current resources
tech-tree-designer analyze --resources 5 --timeline 12

# Update scenario progress 
tech-tree-designer progress --scenario research-assistant --status completed

# Get recommendations prioritizing specific sectors
tech-tree-designer recommend --priority healthcare,education

# Identify current bottlenecks
tech-tree-designer dependencies --bottlenecks
```
