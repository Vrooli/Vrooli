# Date Night Planner

An AI-powered scenario for planning romantic date nights, suggesting activities, restaurants, and itineraries based on user preferences.

## Features
- Personalized date suggestions
- Integration with local resources like calendars and maps
- CLI and API interfaces

## Setup
1. Run `make run` or `vrooli scenario run date-night-planner`

## Testing
Run `make test` or `./test/run-tests.sh`
