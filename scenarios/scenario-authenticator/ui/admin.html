Quick Integration Guide

1. Register Your Application

First, register your scenario using the admin dashboard "+ Register New Application" button above, or generate integration code directly:

scenario-authenticator protect my-scenario api

2. Add Authentication to Your API

For Go APIs, add this validation middleware:

// Add to your protected routes func validateAuth(next http.Handler) http.Handler { return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) { authHeader := r.Header.Get("Authorization") if authHeader == "" { http.Error(w, "Authorization required", 401) return } // Get current API port dynamically apiPort := os.Getenv("AUTH_API_PORT") // Set this in your scenario if apiPort == "" { apiPort = "15837" // fallback to current port } resp, err := http.Get(fmt.Sprintf("http://localhost:%s/api/v1/auth/validate", apiPort)) if err != nil || resp.StatusCode != 200 { http.Error(w, "Invalid token", 401) return } next.ServeHTTP(w, r) }) } // Use middleware: router.Use(validateAuth)

3. Protect Your UI Routes

For JavaScript UIs, add this check (replace ports with your scenario's dynamic ports):

// Check authentication with dynamic ports async function requireAuth() { const token = localStorage.getItem('authToken'); // Get auth service port from your scenario's environment const authApiPort = window.AUTH_API_PORT || '15837'; // Replace with your scenario's port const authUiPort = window.AUTH_UI_PORT || '37578'; // Replace with your scenario's port const response = await fetch(`http://localhost:${authApiPort}/api/v1/auth/validate`, { headers: { 'Authorization': `Bearer ${token}` } }); if (!response.ok) { window.location.href = `http://localhost:${authUiPort}/login?redirect=` + encodeURIComponent(window.location.href); } return await response.json(); // Returns user data }

4. Handle User Context

The token validation API returns this user information:

// GET /api/v1/auth/validate response: { "valid": true, "user_id": "uuid-string", "email": "user@example.com", "roles": ["user"], "expires_at": "2025-01-01T12:00:00Z" } // Invalid token response: { "valid": false }
