# Product Manager Agent

## Overview
The Product Manager Agent is a strategic AI assistant designed to handle big-picture product decisions, feature prioritization, and roadmap planning. It guides coding agents and helps customers make informed product decisions based on data-driven insights.

## Purpose
This scenario transforms Vrooli into an intelligent product management platform that:
- Analyzes market trends and competitor landscapes
- Prioritizes features based on ROI and user impact
- Generates and maintains product roadmaps
- Makes strategic recommendations for product direction
- Processes user feedback and converts it into actionable insights
- Plans sprints and allocates resources effectively

## Key Features
- **Feature Prioritization**: Uses RICE scoring (Reach, Impact, Confidence, Effort) and custom metrics
- **Market Research**: Automated competitor analysis and trend monitoring
- **Roadmap Generation**: Visual, interactive roadmaps with timeline management
- **Decision Analysis**: Data-driven decision trees with risk assessment
- **Sprint Planning**: Agile sprint management with capacity planning
- **ROI Calculation**: Financial impact analysis for feature investments
- **User Feedback Processing**: Sentiment analysis and feature extraction from feedback

## Dependencies
This scenario relies on several other Vrooli capabilities:
- **mind-maps**: For organizing product strategies and feature relationships
- **research-assistant**: For deep market and competitor research
- **task-planner**: For breaking down features into actionable tasks
- **stream-of-consciousness-analyzer**: For processing unstructured product ideas

## UX Design
Professional, clean interface with dashboard-style layout optimized for business users:
- Dark mode with subtle blue/purple accents for a modern corporate feel
- Data visualization charts for metrics and analytics
- Kanban-style boards for feature management
- Timeline views for roadmap visualization
- Clean typography with focus on readability

## Target Users
- Product managers and owners
- Startup founders making product decisions
- Development teams needing strategic guidance
- Business analysts evaluating feature investments
