## Template Variables Reference

When using these templates, replace:

| Variable | Replace With |
|----------|--------------|
| `[Name]` | User's name if known, otherwise remove the greeting name |
| `[Their question topic]` | Brief summary of their question for the subject line |
| `[Answer their specific question here]` | Direct answer to what they asked |
| `[Specific link to relevant doc]` | URL to the most relevant documentation page |

## Tone Guidelines

- **Direct:** Answer the question or state the outcome first
- **Concise:** Avoid unnecessary pleasantries or filler
- **Helpful:** Provide resources even when declining requests
- **Honest:** If we can't help, say so clearly

Avoid:
- Excessive apologizing
- Marketing language or upselling
- Promises we can't keep
- Vague next steps
