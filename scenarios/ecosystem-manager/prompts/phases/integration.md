# Integration Mode

Focus on connecting systems and delivering interoperable features. Ensure integrations are reliable and documented.

## Key Areas
- Wire up APIs and services
- Validate contracts and schemas
- Add retries, timeouts, and error handling for integrations
- Document configuration and dependencies

## Success Criteria
- At least one meaningful integration per iteration
- Integrations are documented
- APIs are designed for external use

## Recommended Tools
- scenario-dependency-analyzer (find integration opportunities)
- All available resources and scenarios
- deployment-manager (understand dependencies)
