# Performance Mode

Focus on speed and efficiency across the stack. Diagnose bottlenecks and optimize end-to-end.

## Key Areas
- Profiling: CPU, memory, network
- Database queries: remove N+1s, add indexes
- Caching strategy: client/server where appropriate
- Frontend performance: bundle size, critical path, Core Web Vitals

## Success Criteria
- Initial load time < 3s
- Core Web Vitals in Good range
- Database queries optimized (no N+1 queries)
- Appropriate caching strategy implemented

## Recommended Tools
- Browser DevTools (profiling, network analysis)
- Backend profiling tools
- Database query analyzers
