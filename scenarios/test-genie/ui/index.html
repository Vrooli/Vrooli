Get started with Test Genie

Quick Start Guide

1. Generate Your First Test Suite

Navigate to the "Generate Tests" page and enter your scenario name. Select test types and configure options, then click "Generate Test Suite".

2. Execute Tests

Once generated, go to "Test Suites" to view and execute your tests. Monitor progress in the "Executions" page.

3. Analyze Coverage

Use the "Coverage Analysis" page to identify gaps and improve test coverage.

4. Create Test Vaults

For complex scenarios, create test vaults with multiple phases for comprehensive validation.

CLI Commands

# Generate test suite test-genie generate my-scenario --types unit,integration --coverage 95 # Execute test suite test-genie execute [suite-id] --type full --environment local # Analyze coverage test-genie coverage my-scenario --depth comprehensive --report # Create test vault test-genie vault my-scenario --phases setup,develop,test # Check system status test-genie status # Get help test-genie help
