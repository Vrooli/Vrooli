# App Debugger PRD

## Overview
Professional debugging platform for Vrooli apps, providing error analysis, performance monitoring, and fix suggestions.

## Features
- Real-time log viewing and error tracking
- Performance profiling and metrics
- Debug session management
- AI-powered issue diagnosis and fixes
- Integration with Postgres, Redis, QuestDB

## Requirements
- API: Go, port 15000-19999
- UI: Node.js, port 35000-39999
- Storage: Postgres for sessions, Redis for cache, QuestDB for time-series

## User Stories
- As a developer, view live errors and logs to debug apps quickly.
- As an operator, monitor performance to identify bottlenecks.
- As a maintainer, use AI suggestions to resolve issues efficiently.
